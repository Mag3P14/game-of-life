# Game of life

###### Controls:

- `SPACE` pause/unpause
- `LEFT ARROW` slow simulation
- `RIGHT ARROW` speed up simulation
- `LEFT CLICK` spawn cell (only while paused)
- `LEFT CLICK + LEFT SHIFT` kill cell (only while paused)
- `R` spawn random cells
- `C` kill all cells
